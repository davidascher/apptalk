What's a Web App?

A Web App is a website that…

…can run outside of a browser

don't assume built-in navigation controls, URL bars...

…can work offline

but it can work better online!

…looks great on a phone

but it can do more on a monitor!

…understands phone interactions:

touch/swipe, share, camera, …

…How do we do that?
